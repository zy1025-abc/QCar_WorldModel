2026.3.24
当前版本不仅能够根据当前状态精准预测下一时刻的物理变化，还突破了单步限制，实现了自回归长线轨迹推演 (Autoregressive Rollout) 能力。
核心成果摘要：
1.高精度单步推演：最终版模型完全修复了早期的垂直尖刺问题，预测曲线与真实物理轨迹高度重合。
2.工业级数据管道：实现了文件级全局洗牌 (File-Level Shuffle)，彻底解决了时序数据的跨文件拼接瞬移 Bug，保证了测试集中包含极具挑战性的 S 弯和倒车路况。
3.极限闭环梦境推演：模型能够脱离未来的真实物理观测，仅凭过去的 20 帧 (0.8s) 惯性，将自己的预测当做真实输入，连续“闭眼”推演未来长达 10 秒 (250步) 的 9D 物理运动轨迹。

核心执行脚本：
1.process_split_data.py (数据提纯车间)
作用：读取 data_raw 中的离散 CSV，进行全局乱序洗牌、过滤 OptiTrack 瞬移脏数据、修正四元数半球反转，最终切分为 [20帧历史 -> 1帧未来] 的滑动窗口，打包为 .pt 张量。
2.train_world_model.py (模型炼丹炉)
作用：直接读取 .pt 内存数据，按 9:1 的比例切分训练/测试集。训练 LSTM 核心网络，并将模型权重与归一化字典保存至 models_saved。
3.evaluate_and_plot.py (基础验证大屏)
作用：使用传统的滑动窗口（Many-to-One）验证模型的单步预测精度，生成长线 9D 大盘图。
4.evaluate_autoregressive.py (🚀 进阶自回归引擎)
作用：测试模型长线生存极限。只需在代码底部修改 PREDICT_W 参数，即可让模型一口气向未来发射 10步、100步 甚至 250步 的预测轨迹，观测累积误差的发散情况。

目录结构说明：
data_processed/：清洗完毕后的终极纯净数据张量库（包含 forward 和 backward 的 .pt 文件），供显卡直接读取。
models_saved/：交付给 CARLA 组的最终版“AI 大脑”，包含训练好的权重 .pth 和归一化特征字典 .pt。
results_evaluation/：标准单步验证的 9D 大盘图和完整的预测结果 CSV 总表（用于进行数据分析的最直观文件）。
results_evaluation_w=.../：不同预测步长的自回归极限生存测试报告：
	w=50：未来 2.0 秒的短程推演图表。
	w=100：未来 4.0 秒的中程推演图表。
	w=250：未来 10.0 秒的长程梦境推演（可极其直观地看到长线物理误差是如何像滚雪球一样发散的）。
old_versions/：包含早期 V1/V2 版本的废弃代码、未洗牌的旧数据和有尖刺的旧图表，仅作历史回溯参考。



2026.3.9
本项目完成了自行车（Q-Car）的 **Digital Twin（数字孪生）预测网络** 开发。该模型基于 **World Model（世界模型）** 理念，通过 25Hz 的真实行驶数据训练，能够根据当前状态预测下一时刻的物理变化。

## 核心成果摘要

*预测精度*：最终版模型训练 Loss 降至 **0.592**（相比初始版本提升约 25%）。
*优化点*：
1. **数据清洗**：成功识别并剔除了 89 条由于原始 CSV 数据拼接导致的“时空断层”瞬移脏数据。
2. **模型扩容**：将 LSTM 隐藏层神经元从 128 提升至 256，并增加至 3 层网络深度。
3. **惯性增强**：时序记忆长度从 10 帧扩展至 **20 帧 (0.8s)**，能更有效捕捉车辆运动惯性。

📁 目录主要内容说明

1. `models_saved/` (**当前最优模型 - 推荐使用**)

这是交付给 CARLA 组直接调用的最终版“AI 大脑”：

*`forward_world_model.pth / backward_world_model.pth`*：最终训练权重。
*`forward_normalization.pt / backward_normalization.pt`*：**数据归一化字典**。输入数据前必须先以此进行平移处理，否则预测会失效。
*特点*：曲线极其平滑，已彻底修复早期版本的预测尖刺问题。

2. `results_evaluation/` (**最终版验证报告**)

*14 个误差对比表格 (CSV)*：记录了位置 (x,y,z) 和姿态 (rot_0-3) 的预测值与真实值。
*对比趋势图 (PNG)*：展示了最终模型输出的曲线，**蓝色实线**（真实轨迹）与**橘色虚线**（AI 预测）几乎完美重合。

3. `results_evaluation_v1/` (**初始版对比报告**)

*价值分析*：在该文件夹的 `pos_z` 或 `rot_3` 图表中，可以看到明显的**垂直尖刺**。
*意义*：通过对比 `v1` 和 `最终版` 的图表，可以直观看到我们通过数据清理技术成功抹平了物理噪音。
